Этот пример демонстрирует drag-and-drop строк между двумя компонентами Grid следующих типов: `DataGrid`, `TreeDataGrid` и `GroupDataGrid`. 
На каждой вкладке расположены два DataGrid, и строку можно перетащить из одного в другой.

Перетаскивание включается атрибутом `rowsDraggable="true"`, а каждый Grid становится целью сброса с `dropMode="ON_GRID"`. Событие `GridDragStartEvent` запоминает перетаскиваемые строки и их исходный контейнер, а слушатель `GridDropEvent` перемещает их в целевой контейнер.

Для `TreeDataGrid` перетаскиваемый узел перемещается вместе со всем его поддеревом и становится корневым узлом в целевом дереве. Для `GroupDataGrid` перемещённые строки автоматически перегруппировываются по колонке группировки.
