`DataGrid.setSortBuilderDelegate()` настраивает способ сортировки колонки.

Пример выше сортирует колонку *Grade* по кастомному бизнес-приоритету (сначала HIGH, затем PREMIUM, затем STANDARD) вместо порядка по умолчанию для перечисления, используя `Comparator`, заданный через `DataGridSortBuilder.replaceSort()`.

Без делегата колонка сортируется по порядку объявления констант (PREMIUM, HIGH, STANDARD), потому что Java-enum реализует `Comparable` и сравнивается по `ordinal()`.

Компараторы применяются для сортировки в памяти — здесь она происходит из-за того, что весь набор данных загружен на клиенте. Для серверной (persistent) сортировки задавайте JPQL-выражение.
