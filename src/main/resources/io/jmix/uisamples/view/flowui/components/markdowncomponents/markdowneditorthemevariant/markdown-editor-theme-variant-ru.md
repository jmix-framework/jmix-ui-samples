Компонент `MarkdownEditor` поддерживает следующие theme variant:

- `toolbar-align-start` — выравнивает кнопки панели инструментов к началу заголовка.
- `toolbar-align-center` — центрирует кнопки панели инструментов и скрывает разделитель.
- `toolbar-align-end` — выравнивает кнопки панели инструментов к концу заголовка и скрывает разделитель.
- `helper-above-field` — отображает вспомогательный текст над полем, под меткой.

Применить эти варианты можно с помощью атрибута `themeNames` в XML-дескрипторе или в контроллере представления с
использованием констант из класса `MarkdownEditorVariant`.
