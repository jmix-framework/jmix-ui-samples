Этот пример демонстрирует базовый API drag-and-drop для произвольных компонентов.

Используйте `DragSource.create()`, чтобы сделать компонент перетаскиваемым, и `DropTarget.create()`, чтобы превратить контейнер в цель для сброса. Обе зоны являются целями для сброса, поэтому карточки можно перетаскивать из одной зоны в другую в любом направлении.

При сбросе карточки `DropEvent.getDragSourceComponent()` возвращает перетаскиваемый компонент, и его добавление в целевую зону перемещает его туда из прежнего родителя.
