Notifications имеет несколько настроек темы и типа:

Варианты темы:

  • primary
  • info
  • success
  • warning
  • error
  • contrast

Типы:

  • default
  • success
  • warning
  • error
  • system

Вы также можете настроить продолжительность уведомления или возможность закрытия.

Все эти настройки доступны через Notifications API.
