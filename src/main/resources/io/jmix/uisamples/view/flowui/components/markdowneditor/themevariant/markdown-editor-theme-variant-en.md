The `MarkdownEditor` component supports the following theme variants:

- `toolbar-align-start` тАФ aligns the toolbar buttons to the start side of the header.
- `toolbar-align-center` тАФ centers the toolbar buttons and hides the divider.
- `toolbar-align-end` тАФ aligns the toolbar buttons to the end side of the header and hides the divider.
- `helper-above-field` тАФ renders the helper text above the field, below the label.

You can apply these variants using the `themeNames` attribute in the XML descriptor or in the view controller using
constants from the `MarkdownEditorVariant` class.
