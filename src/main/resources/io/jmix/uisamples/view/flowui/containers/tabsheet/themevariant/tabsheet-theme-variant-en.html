The TabSheet component has several theme variants:

  • icon-on-top
  • centered
  • small
  • minimal
  • hide-scroll-buttons
  • equal-width-tabs
  • bordered
  • no-padding
  • small
  • hide-scroll-buttons
  • no-border
  • no-padding

You can set these styles to the TabSheet component using the themeNames attribute either in the XML descriptor or in the view controller using constants from TabSheetVariant class.
