`DataGrid.setSortBuilderDelegate()` настраивает способ сортировки колонки.

Алиас `{E}` ссылается на загружаемую сущность. Так можно заставить колонку сортироваться по другому или вычисляемому персистентному выражению, а не потому значению, которое она отображает. Выражение применяется для серверной (persistent) сортировки — здесь она происходит из-за того, что данные постраничные.

Для запроса по `KeyValueEntity` алиас `{E}` не поддерживается — используйте конкретный алиас из запроса, например `"e.lastName"` для запроса вида `select e from Customer e`.
