Компонент `MarkdownEditor` позволяет пользователю редактировать текст с использованием синтаксиса Markdown. Он содержит
вкладки Edit/Preview, панель форматирования с основными действиями Markdown и поддерживает горячие клавиши.

Значением компонента является исходный текст в формате Markdown. Переключитесь на вкладку *Preview*, чтобы увидеть
отрисованный результат.
