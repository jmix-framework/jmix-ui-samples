Компонент Calendar позволяет выводить информацию в виде расписания, компоновать события, вызывать действия для конкретных событий. В данном примере показаны:

  • При указании диапазона значений с помощью атрибутов startDate и endDate более 7 дней, компонент отображает весь месяц целиком.

  • События календаря могу определять имена CSS стилей при помощи атрибута styleName.

  • По клику на событие календаря открывается модальное окно с детальной информацией.
