JavaScriptComponent - это простой компонент пользовательского интерфейса, который представляет собой универсальную обёртку для использования любой готовой библиотеки JavaScript в приложении на базе Jmix напрямую, без необходимости подключать компонент Vaadin.

В этом примере мы используем Wickedpicker - простой jQuery плагин выбора времени.

jsComponent имеет следующие зависимости:

  • jQuerry, подключаемый из WebJar ресурса.
  • файлы плагина и коннектора, расположенные в директории web/VAADIN/timepicker модуля web.
  • Файлы шрифта, необходимые для Wickedpicker и расположенные в директории web/VAADIN/fonts модуля web.
