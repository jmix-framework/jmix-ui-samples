Компонент EntitySuggestionField объединяет в себе функциональность поля с подсказками и компонента EntityPicker, предназначенного для отображения экземпляра сущности. Второй способ связать компонент с данными - это установить значение атрибуту metaclass.

Логика поиска задаётся с помощью интерфейса SearchExecutor. Можно установить следующие параметры:

  • asyncSearchDelayMs - задержка между последней нажатой клавишей и запуском поиска;
  • suggestionsLimit - ограничение количества показываемых подсказок;
  • minSearchStringLength - минимальное количество символов, которое нужно ввести, чтобы запустить.
