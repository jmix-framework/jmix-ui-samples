ScrollBoxLayout - контейнер, который позволяет прокручивать свое содержимое.

С помощью атрибута orientation можно задавать направление расположения вложенных компонентов - horizontal или vertical. По умолчанию vertical.

Атрибут scrollBars позволяет настраивать полосы прокрутки. Он может принимать значения horizontal, vertical - для прокрутки по горизонтали и вертикали соответственно, both - для прокрутки во всех направлениях. Установка значения none запрещает прокрутку в любом направлении.

Вложенные в scrollBox компоненты должны иметь фиксированные размеры или размеры по умолчанию. Нельзя устанавливать height="100%" или width="100%". В то же время scrollBox не может вычислять свои собственные размеры по содержимому. Ему нужно либо указать абсолютные размеры, либо растянуть в родительском контейнере, установив height="100%" и width="100%".
