Если поле не связано с атрибутом сущности (то есть не указан источник данных и название атрибута), то можно указать тип значения поля с помощью атрибута datatype. В качестве значения атрибута может быть указано любое название типа данных, зарегистрированного в метаданных приложения - см. Datatype. Как правило, в TextField используются следующие типы данных:

  • decimal

  • double

  • int

  • long
