В данном примере показано, как можно реализовать кастомный фильтр с помощью декларативного набора условий запроса, связанных с полем ввода фасетом DataLoadCoordinator.

DataLoadCoordinator в автоматическом режиме заставляет также срабатывать загрузчики данных на событии BeforeShowEvent.

Обратите внимание, что компонент универсальный фильтр можно использовать совместно с кастомным фильтром.
