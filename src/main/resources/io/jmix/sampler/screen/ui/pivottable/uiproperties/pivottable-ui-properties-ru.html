В PivotTable имеется возможность не показывать UI элементы в режиме редактирования. Для этого нужно использовать свойство showUI.

Следующие свойства не зависят от режима редактирования, однако работают они только с табличными рендерерами:

  • showRowTotals отображает сумму по строке;
  • showColTotals отображает сумму по столбцу.
