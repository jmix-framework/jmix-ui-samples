Компонент GroupTable - это таблица с возможностью динамической группировки по любому полю. Для того чтобы сгруппировать таблицу по какой-либо колонке, нужно в заголовке таблицы перетащить эту колонку в позицию слева от разделителя в заголовке таблицы.

Для GroupTable в атрибуте container элемента rows должен быть указан groupDataContainer. В противном случае группировка работать не будет.

group - необязательный элемент, может в единственном экземпляре находиться внутри элемента columns. Содержит набор элементов column, по которым будет выполняться первоначальная группировка при открытии экрана.
