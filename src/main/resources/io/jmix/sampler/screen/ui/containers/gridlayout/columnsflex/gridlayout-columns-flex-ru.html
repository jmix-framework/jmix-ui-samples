Для распределения незанятого места неравными долями необходимо определить для каждой колонки элемент column и задать для него атрибут flex. Если атрибут flex не указан, или указано значение 0, то ширина данной колонки будет установлена по содержимому, если хотя-бы одна другая колонка имеет ненулевой flex.

В данном примере первая и третья колонки получат ширину по максимальной длине текста надписей, вторая и четвертая займут все оставшееся место, при этом четвертая в три раза больше чем вторая.

Для того, чтобы лишнее место вообще образовалось, необходимо установить всему контейнеру ширину в пикселах или процентах. В противном случае ширина колонок будет рассчитана по ширине содержимого, и атрибут flex не будет иметь никакого эффекта.
