KeyValue Containers позволяют выполнять запросы, возвращающие скалярные и агрегатные значения. KeyValue Containers работают с сущностями особого типа: KeyValueEntity. Такая сущность может содержать произвольный набор атрибутов, задаваемый во время работы приложения. KeyValue Containers предназначены только для чтения данных, так как сущность KeyValueEntity является неперсистентной и не может быть сохранена стандартным механизмом работы с БД.
