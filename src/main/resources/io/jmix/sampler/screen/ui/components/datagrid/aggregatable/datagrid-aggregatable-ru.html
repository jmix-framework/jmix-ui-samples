Атрибут aggregatable включает режим агрегации строк DataGrid. Поддерживаются следующие операции:

  • SUM - сумма

  • AVG - среднее значение

  • COUNT - количество

  • MIN - минимальное значение

  • MAX - максимальное значение

Для агрегируемых колонок необходимо указать элемент aggregation с атрибутом type, задающим функцию агрегации. Агрегированные значения столбцов выводятся в дополнительной строке вверху DataGrid.

Атрибут aggregationStyle позволяет задать положение строки агрегации: TOP или BOTTOM. По умолчанию используется TOP.

В дополнение к операциям, перечисленным выше, можно задать собственную стратегию агрегации путем создания класса, реализующего интерфейс AggregationStrategy.
