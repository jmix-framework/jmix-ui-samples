Список опций компонента может быть задан с помощью методов setOptionsList(), setOptionsMap() и setOptionsEnum().

  • При использовании setOptionsList() выпадающий список будет отображать результаты преобразования каждой опции методом toString().

  • Метод setOptionsMap() позволяет явно задать строковые названия значениям опций.

  • Метод setOptionsEnum() принимает в качестве параметра класс перечисления. Выпадающий список будет содержать локализованные названия значений перечисления, значением компонента будет являться выбранное значение перечисления.
