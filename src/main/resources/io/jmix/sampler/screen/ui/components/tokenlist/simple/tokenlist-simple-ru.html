Компонент TokenList представляет собой упрощенный вариант работы со списком сущностей: названия экземпляров располагаются в вертикальном или горизонтальном списке, добавление производится из выпадающего списка, удаление - с помощью кнопок, расположенных рядом с каждым экземпляром.

В данном примере определены два источника данных: вложенный источник данных orderItemsDc, содержащий коллекцию входящих в состав заказа строк, и allItemsDc, содержащий коллекцию всех строк заказа, имеющихся в базе данных. Компонент TokenList отображает содержимое источника данных orderItemsDc, а также позволяет изменять эту коллекцию, добавляя в него экземпляры из источника данных allItemsDc.

Элементы tokenList:

  • lookup - описатель компонента выбора значений.
  • button - описатель кнопки добавления значений. Может содержать атрибуты caption и icon.
  • clearButton - описатель кнопки очистки значений. Может содержать атрибуты caption и icon.
