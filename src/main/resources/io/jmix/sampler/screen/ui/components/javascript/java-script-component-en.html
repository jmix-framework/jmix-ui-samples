JavaScriptComponent is a simple UI component that can work with any JavaScript wrapper without Vaadin component implementation. Thus, you can easily integrate any pure JavaScript component in your CUBA-based application.

In this sample we use Wickedpicker - a simple jQuery timepicker plugin.

jsComponent has the following dependencies:

  • jQuerry obtained from the WebJar resource.
  • Plugin files and the connector are placed in the web/VAADIN/timepicker directory of the web module.
  • Font files required for Wickedpicker are placed in the web/VAADIN/fonts directory of the web module.
