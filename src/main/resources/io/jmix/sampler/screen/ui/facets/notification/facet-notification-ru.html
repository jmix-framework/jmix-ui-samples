NotificationFacet - это невизуальный компонент, позволяющий декларативно настроить уведомления в XML-дескрипторе экрана. Уведомления могут быть показаны с помощью вызова соответсвующего метода: show() или с помощью подписки на действие или кнопку по идентификатору:

  • onButton - определяет идентификатор кнопки, которая вызовет показ уведомления;
  • onAction - определяет идентификатор действия, которое вызовет показ уведомления.
