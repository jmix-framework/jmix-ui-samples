GridLayout places components in a grid.

gridLayout elements:

  • columns - a required element, describes grid columns. It should have either a count attribute, or a nested column element for each column.

    In this example, the width of each column, and hence the whole container width, is adjusted by content.

    If the container width is explicitly defined in pixels or percents, free space will be divided between the columns equally.

  • rows - a required element, contains a set of rows. Each line is defined in its own row element. row element should contain elements of the components displayed in the grid's current row cells. The number of components in a row should not exceed the defined number of columns, but it can be less.
