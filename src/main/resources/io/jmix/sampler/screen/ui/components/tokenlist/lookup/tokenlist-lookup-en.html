You can use the following attributes of the lookup element:

  • lookup attribute makes it possible to select items using an entity lookup screen.

  • lookupScreen attribute sets the identifier of the screen used for items selection in lookup="true" mode. If this attribute is not set, screen identifier is selected according to the rules for the entity_lookup standard action.

  • openType attribute defines how the lookup screen will be opened, similar to what is described for the entity_lookup standard action. Default value - THIS_TAB.
