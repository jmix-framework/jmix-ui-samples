В данном примере конфигурация была создана во время разработки и установлена "по умолчанию" с помощью атрибута default. Поэтому конфигурация появляется сразу после открытия экрана.

Вы можете легко производить фильтрацию по связанным сущностям и их атрибутам:

  • выбрав атрибут сущности из группы Атрибуты диалога Добавить условие поиска.
  • выбрав пункт Создать Условие атрибута диалога Добавить условие поиска.
  • определив условие в XML (propertyFilter).

В данном примере список заказов может быть отфильтрован по заказчикам и по уровням заказчиков.
