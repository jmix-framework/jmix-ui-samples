# AGENTS.md

## What this is

`jmix-ui-samples-2` is a public Jmix Flow UI (Vaadin) web app — a catalog of UI samples: components, layouts, dialogs/notifications, custom components, and a "Cookbook" of recipes. Each sample is a mini-view shown with its source and description. Live: https://demo.jmix.io/ui-samples.
